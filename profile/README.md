
---

## 🔌 Plugins

Plugins developed following e107 best practices:

- `plugin.xml` with correct `<pluginPrefs>` namespacing
- Bootstrap 5 templates separated from PHP logic
- PHPDoc on all public methods
- `README.md` + `DEV_NOTES.md` + `CHANGELOG.md` included
- PHP 8.x compatible
